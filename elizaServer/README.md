# Eliza  Server

## Quickstart
Update .env
1. `npm i -g pnpm`
2. `pnpm i`
3. `pnpm build`
4. `pnpm start --character="characters/beanz.character.json"`
5. start another command line `pnpm start:client`

## Usage
Eliza server allows for calls that use the character file as the agent

## Dependencies
`pnpm i`
